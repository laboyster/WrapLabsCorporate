There are 3 types of menu: dark, white and colored respectively. To apply any of these you need to change the navigation class.

  • dark menu: 
            <nav id="navigation" class="navbar navbar-default" role="navigation">
  • white menu: 
            <nav id="navigation" class="navbar navbar-inverse" role="navigation">
  • colored menu: 
            <nav id="navigation" class="navbar navbar-color" role="navigation">

Some menus have by default a colored section. To remove it just had a class remove-left-red like this: 

    <nav id="navigation" class="navbar navbar-color remove-left-red" role="navigation">
Grid System and Single row structure

The default Bootstrap grid system utilizes 12 columns. It's responsive grid and adapts to viewport. Below 767px viewports, the columns become fluid and stack vertically. The fluid grid system uses percents instead of pixels for column widths. Each row always include 12 columns (for example col-md-8 + col-md-4 = col-md-12).

Google Map (GMAP3 Plugin)

This is google map with custom marker, you can define couple parameters:

  • data-location - address to show
  • data-text - text in bubble overlay
  • data-height - map height (if undefined height will be 220px)
  • data-offset - by how many map should be repositioned from marker center point (default -30)

It uses gmap3 plugin.
Documentation: here
HTML Markup: 

   <div class="googleMap" data-location="Junipers Blvd. 380 Queens, NY 11379, USA" data-text="Junipers Blvd. 380 Queens, NY 11379, USA" data-height="480" data-offset="-80"></div>

Contact Form

Template comes with ready to use:

  • Contact Form
  • Newsletter Form

All needed files are in /form directory.
Remember to include <script src="form/js/form.js"></script> on page.
To configure form just edit form/form.php and change e-mails to your own : 

    $to = "mail@example.com";
    $from = "mail@example.com";

Please note that by default NodeJS doesn't handle PHP which is required for sending email messages. In order to make it work, you need to copy "form" directory to any domain/server where PHP is supported. Once copied, just change in the markup below "URL WHERE PHP WILL WORK" to this domain.

That's it!

How to add more fields to form ?

Just add another form-group section,
for example additional phone and fax : 

    <div class="form-group">
      <label>Your Phone</label>
      <input type="text" class="form-control" name="field_[]" placeholder="enter your phone">
    </div>
    <div class="form-group">
      <label>Your Fax</label>
      <input type="text" class="form-control" name="field_[]" placeholder="enter your fax">
    </div>
